This module provides a function to convert prefix notation domains to infix notation domains.

This could be useful to understand prefix notation because most people are more used to infix notation.

**Table of contents**

.. contents::
   :local:

Usage
=====

>>> from odoo.addons.base_domain_infix_operator.infix_expression import to_infix_domain
>>>
>>> prefix_domain = ["!", "&", ("a", "=", "b"), "!", ("c", "=", "d"), "|", ("e", "=", "f"), "!", ("g", "=", "h")]
>>> to_infix_domain(prefix_domain)
[(('NOT', (('a', '=', 'b'), 'AND', ('NOT', ('c', '=', 'd')))), 'AND', (('e', '=', 'f'), 'OR', ('NOT', ('g', '=', 'h'))))]
