This is a technical module. Because you might want to block users from
writing on a record according to custom logic, this module allows you to
do that in a succinct way.

**Table of contents**

.. contents::
   :local:

Usage
=====

Inherit the mixin in your code and overload \_get_forbid_write_fns with
the appropriate values and implement the check functions.
