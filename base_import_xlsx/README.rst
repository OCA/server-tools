This module adds back support for excel xlsx files in situations where the
installed version of xlrd is 2.0 or above. For these situations xlrd is replaced
by the modern openpyxl library.

Known issues / Roadmap
======================

It is expected that in future versions of Odoo support for xlsx files will be fully
restored. However, as Odoo will not ontroduce new libraries in stable versions,
support is not expected for Odoo 15.0 and before.
