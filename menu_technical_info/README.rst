===================
Menu Technical Info
===================

This module extends the debug mode. When debug mode is enabled the XML ID is shown while the cursor is hovering over a menu item.

Usage
=====

#. Click on 'About';
#. click on 'Activate the developer mode';
#. hover over any menu item.
