======================
User Password Settings
======================

This module allows an admin to manage Password Strength rules, which would be enforced on next password change.

Installation
============

To install this module, you need to:

* Nothing special

Configuration
=============

To configure this module, you need to:

* Go to Settings->Configuration->General Settings and setup the Password Strength section

 * Minimum Password length in character
 * Use Capital Letter
 * Use digit
 * Use special character


Usage
=====

This module will enforce the selected rules on password change.
