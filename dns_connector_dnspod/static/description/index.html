Dnspod Connector Management

Allows to manage your DNS domains through Odoo and www.dnspod.com thanks to odoo-connector .

Create backend link to DNSpod domain server

Create a backend for www.dnspod.com in Odoo to allow the synchronization between Odoo and DNS provider.


The second step is to create the DNS record you want to populate to the backend.

Please note that the connector is split in 2 modules:
- DNS connector to allow generic DNS management
- DNSpod connector which allows specifically to connect to the www.dnspod.com API

Create DNS records in domain


Use Odoo to create your DNS records

Thanks to th odoo-connector technology, a job is enqueud with priority and will directly create the record in DNSPod

Exception handling




When a DNS record created in the odoo is not sync'ed properly with dnspod (API exception), the record state will be changed to Exception. If successfull, the state will display "Done".

DNS Record editable tree view

You can modify record directly the DNS and records through an editable tree view..
