This module adds the extra unidecode option to be applied to the
exported data. Example: e.g. ESPAÑA to ESPANA or forçut to forcut.

**Table of contents**

.. contents::
   :local:

Installation
============

To install this module, you need to install following python library,
**unidecode**.

Usage
=====

To apply unidecode just check the Unidecode field for the data where you
want to apply it in the Excel template. If you are defining the template
you only have to specify @?unidecode? e.g. 'A2':
'picking_id.export_reference${value or ""}@?unidecode?'
