=====================================
Incoming Mail Embedded Picture Reader
=====================================

Mail having embedded pictures will be correctly rendered into Odoo.
Currently this module depends on the Odoo PR https://github.com/odoo/odoo/pull/10435

Configuration
=============

To configure this module, you need to:

 * Configure an incoming/outgoing mail server

Usage
=====

To use this module, you need to:

 * Be able to send/receive email into your Odoo
 * Send a mail with Odoo to a partner
 * Reply this email by adding an embedded picture into the body
 * Fetch the mail into Odoo
 * Embedded picture will be available into the body of the mail
