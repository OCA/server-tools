====================================
Employees Synchronize Gravatar image
====================================

Extends Synchronize Gravatar image to employees

Installation
============

To install this module, you need to:

* base_user_gravatar

* hr

Usage
=====

To configure this module, you need to:

* go to user profile
* press "Get Gravatar image" button
